# To run the whole application

- run this command: \
  `npm run dev-start`

# To run the server

- install packages by this command \
  `npm install`

- run this command \
  `npm start`

# APIs docs

## signup:

- route name: api/user
- body: name, email, password, confirm_password, birthdate

## login:

- route name: api/login
- body: email, password

## login:

- route name: api/blog

* header: auth-token

- body: title, body, category
